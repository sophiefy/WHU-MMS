# WHU-MMS

## 介绍
数据库大作业之资料管理系统（materials management system）。

管理端功能：

- 图书管理（增删查改）
- 论文管理（增删查改）
- 用户管理（增删查改）
- 统计功能

用户端功能：

- 图书查询与购买
- 论文查询与上传
